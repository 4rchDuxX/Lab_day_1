# 🔐 AppSec Training Lab — Den 1

Praktické laby ke školení **Hacking & Bezpečnost webových aplikací**.

## Jak spustit prostředí

1. Klikni na zelené tlačítko **Code** (vpravo nahoře)
2. Vyber záložku **Codespaces**
3. Klikni **Create codespace on main**
4. Počkej ~2 minuty — prostředí se automaticky nainstaluje
5. Otevře se VS Code v prohlížeči s terminálem připraveným k použití

> ✅ Není potřeba nic instalovat na vlastní počítač.

## Co je nainstalováno v prostředí

- `nmap` — síťový skener
- `tshark` — Wireshark v příkazové řádce
- `trufflehog` — skener secrets v Git repozitářích
- `wget`, `curl` — síťové utility
